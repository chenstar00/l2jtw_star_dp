L2J DataPack Project
==============

L2J Server 是一款完全以 Java 編寫的開源伺服器模擬器，適用於著名的韓國 MMORPG。
--------------

本頁面僅提供非常基本的信息，有關構建和開發 L2J Server 的詳細信息，請訪問開發者論壇。

- master 分支是最新的 High Five 版本

- 開發分支是目前的高五開發

使用 l2jserver HighFive 版本更新

使用10幾年前的 l2jtw 中文化來做編譯版本 

兩者融合成為 l2jstar 版本
